# 🍔 Restaurant Ordering App

A simple restaurant ordering application built with **HTML, CSS, and JavaScript** as part of the **Scrimba Fullstack Developer Path**.

The application allows users to browse a restaurant menu, add and remove meals, calculate the total price dynamically, and complete an order through a payment modal.

---

## ✨ Features

- 🍕 Dynamically renders all menu items from a JavaScript array.
- ➕ Add meals to the order with a single click.
- ❌ Remove individual items from the order.
- 💰 Automatically calculates the total price.
- 🛒 Supports ordering multiple quantities of the same meal.
- 💳 Complete the order through a payment form.
- 🎉 Displays a personalized success message after payment.
- ⚡ Updates the UI instantly whenever the order changes.
